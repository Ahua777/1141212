# 好讀周報 - 大麥克經濟學

## 專案簡介
本專案為「好讀周報」的網頁版文章，主題為**大麥克經濟學 (The Big Mac Index)**。內容探討不同國家的大麥克價格差異，並介紹「購買力平價」(Purchasing Power Parity, PPP) 的經濟學概念，以及如何利用數據進行趨勢分析與預測。

## 內容摘要
*   **購買力平價 (PPP)**：解釋一價定律與匯率的關係。
*   **大麥克指數**：利用麥當勞大麥克的價格來衡量各國匯率是否被高估或低估。
*   **趨勢分析**：透過歷年價格數據，計算趨勢方程式來預測未來價格。
*   **經濟學盲點**：探討運用大麥克指數時可能忽略的成本結構、文化差異等因素。
